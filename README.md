# WhatsApp Chatbot

## Overview
This project is a WhatsApp chatbot that utilizes the Meta API to interact with users seamlessly and effectively. The bot is designed to understand and respond to various types of questions by leveraging Large Language Models (LLMs), providing informative and accurate responses to users. 

## Features
1. **Meta API Integration**: The chatbot connects with WhatsApp through Meta's API, which ensures reliable message delivery and receipt.
2. **Webhook Verification**: The application implements webhook verification to secure message handling. This ensures that incoming messages are from verified sources, maintaining the integrity and security of the bot's interactions.
3. **Message Interactions**: Users can interact with the chatbot through various messages. The bot is programmed to understand and categorize these messages to provide appropriate responses.
4. **LLM Responses**: By integrating a large language model, the chatbot can handle complex queries and provide well-structured responses to users, covering a wide range of topics.
5. **Handling Various Question Types**: The bot is capable of addressing different types of questions, including general knowledge, FAQs, and user-specific queries, making it versatile in terms of user interaction.

## Installation
To set up the WhatsApp chatbot locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Satharva2004/whatsapp-chatbot-backend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd whatsapp-chatbot-backend
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To run the chatbot, execute the following command:
```bash
node server.js
```
